Difference between Fetch and Pull :

	Fetch : fetch  only shows in which files changes and what changes are made. It do not merge the changes in the local repo only shows the changes.

	Pull : pull brings all the changes made in remote repo. It merges all the changes made in remote repo to local repo.



Difference between stage the changes and stash the changes :

	Stash : stash are used to save the uncommited changes for later use, later can re-apply stash.  

	Stage : it is step to make before commit. we can modified files to the staging area to create the next commit.
